iNSAN

Interactive Statistical Analysis (iNSAN) is a platform designed to help users understand and process data in an easy and interactive way. Its purpose is to simplify the decision-making process through clear and engaging data visualizations.

iNSAN provides eight main modules that can be used for various purposes, ranging from data clustering, exploring relationships between data, making predictions, identifying similar data, to analyzing WhatsApp messages and news content.
